# Customer Support Chatbot 🤖

An AI-powered chatbot built with Python, NLP, and TensorFlow/Keras to automate 
customer query resolution and reduce manual response time.

## Features
- Intent recognition to classify customer queries
- Automated response generation
- Reduces manual support effort significantly

## Tech Stack
- Python
- NLP (Natural Language Processing)
- TensorFlow / Keras
- NLTK

## How to Run
1. Clone the repo
2. Install dependencies: pip install -r requirements.txt
3. Run: python chatbot.py

## Project Structure
- chatbot.py — Main chatbot logic
- intents.json — Training data (intents & responses)
- train_model.py — Model training script
- requirements.txt — Dependencies
